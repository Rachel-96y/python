注意： 目前这种打包方式仅适用于单个.pyd文件  (即源文件)
#  TEXT_V_0.03这个文件夹里的例子是利用:"仓库管理系统"来演示的
#  具体就是将源文件经由cython转为c/c++代码后再编译为*.pyd动态链接库文件
#  这样做的目的是保护源码不被泄露
#  做好后的样品都在software文件夹目录下,最终的软件在dict中(暂时非单个文件)
#  使用已经做好的小工具(setup.pyw)可以将*.py文件经由cython转为*.pyd的动态链
接库文件并在删除生成的*.c文件后将*.pyd文件统一放入software文件夹下
#  需要使用run.bat执行特定的命令去运行setup.pyw
#  需要一个入口程序(它将不被编译成*.pyd)来启动程序
#  其它的文件结构和逻辑不变
